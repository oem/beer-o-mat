## What is it?

An image classifier, trained on beer labels.

Check out the [article on medium](https://medium.com/p/243572a79d) for an in-depth description.

## usage

### setup

install the requirements: `pip install -r requirements.txt`

### label data

`make prepare_data`

### train the model

`make train_model`

### test

`make test`
